   - SSL 인증서 확인 필요

## 3. 개선 사항

### 3.1 완료된 개선사항 ✅
1. **조건부 임포트 추가**
   - PyQt6 없이도 기본 기능 테스트 가능
   - numpy 없이도 모듈 로드 가능
   - ffmpeg-python 없이 subprocess 폴백

2. **시뮬레이션 모드**
   - NDI 없이도 기본 동작 가능
   - 테스트 환경에서 유용

3. **모듈화**
   - 각 관리자가 독립적으로 작동
   - 의존성 분리

### 3.2 필요한 개선사항
1. **설정 파일 경로 수정**
```python
# pd_app/config/settings.py 수정 필요
if not self.config_file:
    self.config_file = "config/settings.json"
```

2. **에러 핸들링 강화**
   - 네트워크 오류 시 graceful degradation
   - 사용자 친화적인 오류 메시지

3. **로깅 개선**
   - 디버그 모드 추가
   - 로그 레벨 설정
